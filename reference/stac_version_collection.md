# Version Collection node (parent of the dataset Collections)

Version Collection node (parent of the dataset Collections)

## Usage

``` r
stac_version_collection(ds_keys, cfg)
```

## Arguments

- ds_keys:

  character vector of dataset keys

- cfg:

  config from
  [`stac_cfg()`](http://marinesensitivity.org/msens/reference/stac_cfg.md)

## Value

STAC Collection as a list
