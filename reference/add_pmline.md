# Add PMTile outline line layers to a map

Add PMTile outline line layers to a map

## Usage

``` r
add_pmline(m, outlines)
```

## Arguments

- m:

  map or map_proxy

- outlines:

  list of specs, each with: url, source_layer, and optionally id,
  source_id, line_color, line_width, line_opacity, filter, before_id

## Value

map (pipeable)
