# Dataset Collection node

Dataset Collection node

## Usage

``` r
stac_dataset_collection(ds, cfg, item_files = character())
```

## Arguments

- ds:

  one-row data.frame from the `dataset` table

- cfg:

  config from
  [`stac_cfg()`](http://marinesensitivity.org/msens/reference/stac_cfg.md)

## Value

STAC Collection as a list
